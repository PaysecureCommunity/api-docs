# Headers

Please specify the following headers in each of the API calls:

```http
Content-Type: application/json
Authorization: Bearer {Merchant API Key}
```

## Authorization

Use the **Bearer Token** method for authentication:

```http
Authorization: Bearer {{APIKey}}
```

Ensure that every request includes these headers for proper authentication and data formatting.

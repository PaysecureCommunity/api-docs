# Environments

These are the BaseUrls for the APIs on different environments:

| Environment | Base URL |
| ----------- | -------- |
| Production  | [https://api.paysecure.net/api/v1](https://api.paysecure.net/api/v1) |
| Sandbox     | [https://api.paysecure.net/api/v1](https://api.paysecure.net/api/v1) |

**Note:** Set the Sandbox flag to true (checked) in your merchant account.
